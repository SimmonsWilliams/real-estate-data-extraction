# Real Estate Data Extraction System

## Overview

This project extracts structured real estate
listing data from public property listing pages.

The scraper collects:

- Property titles
- Prices
- Beds
- Baths
- Sqft
- Property URLs

The extracted data is cleaned and exported into:

- CSV
- Excel
- MySQL Database

---

## Technologies Used

- JavaScript
- DOM Parsing
- Playwright
- MySQL
- Excel / CSV

---

## Features

- Property listing extraction
- Pagination handling
- Regex-based parsing
- Structured CSV export
- Database import support
- Data cleaning and normalization

---

## Output Example
